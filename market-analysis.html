Methodology & Data Sources

ACS data: US Census Bureau American Community Survey 5-Year Estimates (B01003, B25003, B25004, B25070, B25064). LIHTC: HUD LIHTC Database (public). Geographies: Census TIGERweb tract centroids. All data is pre-computed and bundled; no API key required. See MARKET_ANALYSIS_METHOD.md and PMA_SCORING.md for full methodology.
