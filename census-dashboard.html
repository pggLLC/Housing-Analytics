How this helps multifamily analysis

  • Supply composition. Higher 5+ unit shares typically correlate with stronger apartment market depth and rent elasticity for new product.
  • Site selection. Compare counties or places for multifamily penetration, renter share, and cost burden before committing to a PMA.
  • LIHTC context. Pair structure mix with existing CHFA LIHTC + preservation inventory and the QCT/DDA share to size the affordable-housing pipeline.
  • Vintage check. Year-built distribution flags jurisdictions where preservation/rehab capital may be more impactful than ground-up development.

Data sources & variables

The page reads from two cached snapshot files that are refreshed nightly by CI. No live Census API calls are made from the browser (api.census.gov now requires an API key for all requests).

  • data/census-multifamily-co.json — ACS 5-year DP04 shares for state + 64 counties + 474 places (built by scripts/build_census_multifamily_co.py).
  • data/multifamily-inventory-co.json — affordable-housing inventory rolled up by county (CHFA LIHTC, preservation, HUD MF, USDA RD). Built by scripts/build_multifamily_inventory_co.py.

Census variables surfaced

  • DP04_0001E — Total housing units (estimate)
  • DP04_0007E–DP04_0014E — Units in structure counts (1-detached, 1-attached, 2, 3-4, 5-9, 10-19, 20+, mobile)
  • DP04_0017E–DP04_0026E — Year built (10 decade cells)
  • DP04_0046PE/DP04_0047PE — Owner-occupied / renter-occupied share
  • DP04_0089E — Median home value
  • DP04_0134E — Median gross rent
  • DP04_0137PE–DP04_0142PE — GRAPI rent burden distribution
