‚ôø Accessibility: Always include a <caption> (use .sr-only to visually hide). Use scope="col" on <th> elements. Wrap in a scrollable role="region" container with tabindex="0" so keyboard users can scroll horizontally. Sortable columns need aria-sort updated via JavaScript.
