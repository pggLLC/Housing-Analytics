# `js/site-search.js`

js/site-search.js — site-wide search (powers search.html).

Loads the build-time `search-index.json` (generated by scripts/build-public-site.mjs
from every public page's title, description, and headings) and renders ranked matches.
Uses window.APP_BASE_PATH (from path-resolver.js) so result links resolve root-absolute
and work identically from the root and from /places/ pages. Fully self-contained and
guarded — if anything is missing it no-ops rather than throwing.

_No documented symbols — module has a file-header comment only._
