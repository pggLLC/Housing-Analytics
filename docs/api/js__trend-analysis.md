# `js/trend-analysis.js`

trend-analysis.js
Colorado Housing Trend Analysis Module

Loads four state-level housing metrics for Colorado + 11 peer states
(2019–2024) from data/market/state-trend-analysis.json. That JSON is
generated by scripts/fetch_state_trend_analysis.py, which pulls each
series directly from FRED's public CSV endpoint (no API key) and
aggregates to annual snapshots.

Sources behind each metric:
  • Median Listing Price ($k)     — Realtor.com via FRED  (MEDLISPRI{state})
  • FHFA HPI YoY Growth (%)       — FHFA via FRED         ({state}STHPI, Q4-over-Q4)
  • Active Listings (k units)     — Realtor.com via FRED  (ACTLISCOU{state})
  • Private Building Permits (k)  — Census BPS via FRED   ({state}BPPRIV)

Until the JSON file is loaded, the module renders an empty-state
notice with a refresh hint. Re-run the fetcher annually after FRED
publishes Q1 revisions.

Usage: TrendAnalysis.init()  (call after DOMContentLoaded)
Renders into: #trend-analysis-section

## Symbols

### `mean(arr)`

Compute mean of an array.

### `stddev(arr)`

Compute sample standard deviation.

### `welchT(a, b)`

Two-sample Welch t-test statistic.
Returns { t, significant } where significant = |t| > 2.0 (~p<0.05 heuristic).

### `loadData()`

Fetches state-trend-analysis.json and populates METRICS[i].data with
a per-state map keyed by state name. Returns true on success.

If the fetcher partially failed (some series are null), the missing
values stay null; render paths must guard against null entries.
