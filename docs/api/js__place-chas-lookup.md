# `js/place-chas-lookup.js`

place-chas-lookup.js

Browser-side helper for the TIGER-derived place-level CHAS data
produced by the TIGER spatial-join arc:
  PR-C1 — tract-level CHAS aggregations
  PR-C2 — TIGER place→tract spatial membership
  PR-C3 (this) — place-level CHAS via area-weighted apportionment

The dashboard previously inherited the place's "primary county"
CHAS rates, which was wrong for the 26 cross-county jurisdictions
(Aurora, Erie, Longmont, etc.). With this module, place-level CHAS
comes from summing the underlying tracts' CHAS counts weighted by
what fraction of each tract sits inside the place — accurate for
any place regardless of how many counties it spans.

Public API
----------
  window.PlaceChas.init() — fetch + cache the data file
  window.PlaceChas.lookup(geoid) — returns place CHAS or null
  window.PlaceChas.compareToCounty(placeGeoid, countyChasRecord)
    — returns {place, county, delta} for cost-burden rates so a
      caller can show "Erie's renter cost-burden rate is X% vs Weld
      County's Y%". Useful for surfacing the cross-county-line
      impact in plain language.

Data source
-----------
  data/hna/place-chas.json (regenerated by
  scripts/hna/build_place_chas.py).

## Symbols

### `resolveAlias(geoid)`

Resolve a phantom geoid to its canonical TIGER GEOID. Returns the
 input geoid unchanged if no alias entry exists. Always returns a
 zero-padded 7-digit string.

### `lookup(geoid)`

Lookup place CHAS by 7-digit place geoid. Phantom geoids are
 resolved to their canonical TIGER twin before lookup. Returns null
 if not present.

### `compareToCounty(placeGeoid, countyChasRecord)`

Given a place geoid + a county CHAS record (from data/market/chas_co.json),
 return a comparison object useful for surfacing the cross-county-line
 impact:
   {
     place:  { renter_cb30, renter_cb50, owner_cb30, owner_cb50 },
     county: { renter_cb30, renter_cb50, owner_cb30, owner_cb50 },
     delta:  { renter_cb30_pp, renter_cb50_pp, owner_cb30_pp, owner_cb50_pp }
   }
 pp = percentage-point difference (place rate − county rate).

### `formatComparison(comparison, placeName, countyName)`

Format a comparison result as a short HTML banner.
