# `js/data-connectors/bridge-market-summary.js`

js/data-connectors/bridge-market-summary.js
Bridge Data Output — Colorado regional market summary connector.

Loads the cached file data/market/bridge_co_market_summary.json, which is
generated by a GitHub Actions workflow that pulls from the Bridge public API.
Provides regional land-cost and market-velocity context for LIHTC feasibility
and housing-needs analysis.

The cache file is regenerated by the zillow-data-sync workflow.  If the file
is absent, a warning is logged and all methods return null / empty results
gracefully — callers should check isAvailable() before consuming data.

Usage:
  BridgeMarketSummary.load().then(function () {
    var ctx = BridgeMarketSummary.getLandCostContext(39.74, -104.99);
    // ctx.tier → "high"
    // ctx.note → "High land costs — typically >$150K/parcel. ..."
  });

Public API (window.BridgeMarketSummary):
  load()                          → Promise<object|null>
  isAvailable()                   → boolean
  getSummary()                    → object|null
  getStatewide()                  → object|null
  getRegions()                    → object[]   (sorted by name)
  getRuralRegions()               → object[]
  getRegionNearPoint(lat, lon)    → object|null
  getLandCostContext(lat, lon)    → { tier, medianLandValue, regionName, isRural, note }
  getMarketVelocity(lat, lon)     → { transactionCount, priceTrendPct, label }

## Symbols

### `REGION_CENTROIDS`

Approximate geographic centroids for each region key.
Used by getRegionNearPoint() to match a lat/lon to the closest region.
Format: [latitude, longitude]

### `_summary`

@type {object|null} Full parsed JSON from the cache file.

### `_available`

@type {boolean} True once the file has been fetched successfully.

### `_loadPromise`

@type {Promise|null} Reused promise so load() is idempotent.

### `_toRad(d)`

Convert degrees to radians.
@param {number} d
@returns {number}

### `_haversine(lat1, lon1, lat2, lon2)`

Haversine great-circle distance in miles between two lat/lon pairs.
@param {number} lat1
@param {number} lon1
@param {number} lat2
@param {number} lon2
@returns {number}

### `load()`

Fetch the cached market summary JSON.
Uses DataService.getJSON when available, otherwise falls back to fetch().
Safe to call multiple times — returns the same promise on repeat calls.

@returns {Promise<object|null>} Resolves with the parsed summary or null.

### `isAvailable()`

Returns true if the cache file has been fetched and parsed successfully.
@returns {boolean}

### `getSummary()`

Returns the full parsed JSON object, or null if not yet loaded.
@returns {object|null}

### `getStatewide()`

Returns the statewide stats object, or null if not loaded.
@returns {object|null}

### `getRegions()`

Returns an array of all region objects, sorted ascending by name.
Each object is the raw region record from the cache, augmented with
a `key` property set to its key in the regions map.

@returns {object[]}

### `getRuralRegions()`

Returns only regions where is_rural === true, sorted by name.
@returns {object[]}

### `getRegionNearPoint(lat, lon)`

Finds the region whose centroid is closest to the given lat/lon.
Returns the matching region data object (with `key` property), or null.

@param {number} lat
@param {number} lon
@returns {object|null}

### `getLandCostContext(lat, lon)`

Returns land-cost context for the region nearest to the given lat/lon.

@param {number} lat
@param {number} lon
@returns {{ tier: string, medianLandValue: number|null, regionName: string|null,
            isRural: boolean, note: string }}

### `getMarketVelocity(lat, lon)`

Returns market-velocity metrics for the region nearest to the given lat/lon.
Label is "active" (>= 50 transactions), "moderate" (>= 20), or "quiet" (< 20).

@param {number} lat
@param {number} lon
@returns {{ transactionCount: number|null, priceTrendPct: number|null,
            label: string }}

### `_withKey(regionObj, key)`

Returns a shallow copy of the region object with a `key` property added.
Avoids mutating the original cached object.

@param {object} regionObj
@param {string} key
@returns {object}
