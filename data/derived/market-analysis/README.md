# Derived Market Analysis Data

Files in this directory are generated by scripts in `scripts/market-analysis/`.

## Files
- `market_demand.json` — ACS-derived demand metrics (run `build_market_demand.py`)
- `subsidy_layers.json` — HUD LIHTC aggregates (run `build_subsidy_layers.py`)
- `site_opportunities.json` — Top-ranked tract opportunities (run `build_site_opportunities.py`)
- `neighborhood_access.json` — OSM amenity access scores (run `build_neighborhood_access.py`)

## Rebuild
```bash
python scripts/market-analysis/build_market_demand.py
python scripts/market-analysis/build_subsidy_layers.py
python scripts/market-analysis/build_site_opportunities.py
python scripts/market-analysis/build_neighborhood_access.py
```
